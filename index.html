Fuente de Datos
Los datos se cargan automáticamente desde consolidado.json.gz
⚙️ Cómo actualizar los datos
1️⃣
Ejecutar script de producción
Correr consolidado_produccion.py para generar consolidado.json
2️⃣
Comprimir el JSON
Ejecutar python comprimir_consolidado.py → genera consolidado.json.gz
3️⃣
Publicar el .gz
Subir consolidado.json.gz a la raíz junto con index.html. El consolidado.json debe estar en .gitignore
📋 Columnas del consolidado
Comunes: ID_TA · ID_FO · TIPO_ACTIVIDAD · ESTADO_ACTIVIDAD · FECHA_FINALIZACION · VALOR_TOTAL · GERENCIA
Cirion: VALOR_OBRA_CIVIL · VALOR_MATERIAL · VALOR_ADICIONAL · TIPO_NEGOCIO · ZONA · BASE_OPERATIVA · ITEM_FACTURACION
Liberty: VALOR_APROBADO · VALOR_OBRA_CIVIL · COORDINADOR · CIUDAD · CENTRO_DE_COSTOS · BASE_OPERATIVA
Banca: VALOR_ENTIDAD · VALOR_RED · FORMA_ATENCION · TIPOLOGIA · TECNICO · RED_ASOCIADA
